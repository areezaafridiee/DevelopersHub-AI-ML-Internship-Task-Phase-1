# AI/ML Engineering Internship Projects

## DevelopersHub Corporation – AI/ML Engineering Internship

This repository contains projects completed as part of the AI/ML Engineering Internship at DevelopersHub Corporation. The internship focuses on developing practical skills in data analysis, machine learning, predictive modeling, prompt engineering, chatbot development, and model evaluation using real-world datasets.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* yfinance
* Jupyter Notebook
* Hugging Face Transformers
* OpenAI API / Open-Source LLMs
* Git & GitHub

### Objective

Learn how to load, inspect, and visualize a dataset to understand data trends and distributions.

### Dataset

### Skills Demonstrated

* Data loading and inspection using Pandas
* Descriptive statistics and exploratory data analysis (EDA)
* Data visualization using Matplotlib and Seaborn
* Identifying distributions and outliers
